# PalindroneCheckerApp
A simple and efficient application that checks whether a given word, sentence, or number is a palindrome.

A palindrome is a sequence that reads the same forward and backward (ignoring case, spaces, and punctuation if configured).

🚀 Features

✅ Check single word palindromes

✅ Check sentence palindromes (ignores spaces & punctuation)

✅ Case-insensitive validation

✅ Numeric palindrome support

✅ Special character handling option

✅ Real-time validation (optional)

✅ Responsive and user-friendly UI

✅ Error handling for empty input

✅ Efficient O(n) algorithm
